# Atom

Языки/Languages: <br>
[Англиский](README.md) <br>
Русский

**Этот проэкт сделан с помощью ИИ**

Как запустить:
1. Установите [Arial](ARIAL.TTF) шрифт (если вы на Linux и у вас нет его)
2. Запустите main.py в папке проэкта

Скомпилировать (на Linux):

1. Установите Python 3.11 через Wine в путь `C:\Python311`
2. Запустите [build_on_linux_for_windows.sh](build_on_linux_for_windows.sh)
3. Скомпилированный проект в папке `dist`
